# ACAO for all sites
Some sites let any site send it requests rather than maintaining a proper list of allowed sites.

### Detection
App responds with
`Access-Control-Allow-Origin: <url>`
`Access-Control-Allow-Credentials: true`
for every origin url. This means any domain can send it requests, and all of them can include session cookies.


# Whitelisting sites

Only allowed sites get that response back.
Flaws can happen in this -> checking for prefix, suffix or regex only

### Null Origin Value
Some browsers send in a `null` value for the Origin header. Usually its for cross-origin redirects, requests from serialized data, `file:` protocol and sandboxed cross-origin requests.

# Using iframes to create sandboxed requests
```HTML
<iframe sandbox = "allow-scripts allow-top-navigation allow-forms" srcdoc = "<script>//code</script>">
</iframe>
```
