Excessive agency is when the LLM has access to APIs with sensitive info and can be persuaded by an attacker to perform malicious tasks using said APIs.

Try simply asking what APIs are accessible. Then try variations with random stuff like claiming you're a developer, etc. Try to get as much info as possible with this.

##### Chaining Vulns

Some APIs may look harmless. LLMs can execute other attacks on APIs ( classic web attacks )

A common way for RCE is if $(*cmd*) is accepted.
