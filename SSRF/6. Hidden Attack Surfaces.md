# Partial URLs
Requests might contain URL segments that are joined server-side to make a full URL. Look for recognizable hostnames/paths. But working with this can be tough as server-side processing is unknown.

# URLs in Data Formats
XML, JSON, etc. might be used to transfer URLs, which might be vulnerable to injection

# Referer Header
Some server-side analytics might use `Referer` header to visit unrecognised origins.
