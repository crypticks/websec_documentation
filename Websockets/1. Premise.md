Websockets is a technology that allows bidirectional simultaneous connection.
Usually used for chats, calls etc.

Think of it as an upgraded version of HTTP.
HTTP is like sending letters, Websockets is like a phone call.

An HTTP handshake requests for and confirms a websocket between client and server.
They exist long-term.

Burp Suite can intercept, modify, repeat Websocket packets and so on.
The `Intercept` tab shows Websocket history.
The `Repeater` can modify and resend packets.

Capturing the handshake is also essential during attacks.

Every injection vulnerability is possible with Websockets too.
